Git fue creado por Linus Torvalds en 2005, para ser utilizado en el desarrollo del kernel de Linux.
GitHub le pertenece desde el 2018 a Microsoft, porque al ser el mayor repositorio de código del mundo es punto de encuentro para millones de desarrolladores, y con el control de GitHub, Microsoft se hace también con la comunidad de desarrollo más nutrida del planeta.
Si hay formas de trabajar con GitHub que no sea desde la terminal, por ejemplo hay programadores que utilizan TortoiseGit.
